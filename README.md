# AI News Video Generator (Task 1)
Pipeline: Trending news scraping → script generation → image retrieval → voiceover → video generation
Outputs: Generated videos available in output/videos/ + report PDF included.
## Setup
1. Clone the repo
2. Create `.env` file using `.env.example`
3. Install requirements:
   pip install -r requirements.txt
4. Run:
   python main.py
